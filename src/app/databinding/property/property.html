Property Binding

Property binding in Angular allows you to bind a component property to an element property. It is a one-way binding from the component to the view.

Syntax: [property]="expression"

In this syntax, property is the name of the DOM property you want to bind to, and expression is a JavaScript expression that evaluates to the value you want to assign.

Property binding is used to set the value of an element property, affecting the behavior or appearance of the element.

Property Binding

Examples

<img [src]="imageSource" [alt]="imageAlt">

In this example, the src and alt properties of the <img> element are bound to component properties.
