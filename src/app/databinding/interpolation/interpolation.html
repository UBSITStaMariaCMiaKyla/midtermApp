String Interpolation

Angular string interpolation is a one-way data binding technique that allows you to display dynamic data from your component's TypeScript code within your HTML template. It uses the double curly braces syntax {{ curly }}.

In Angular, string interpolation (also known as text interpolation) is a one-way data-binding technique used to display dynamic data from your TypeScript component in your HTML template.

  1. Basic Characteristics

    2. You embed expressions within double curly braces. Angular evaluates the expression and replaces it with the resulting string in the view.

    • Component Property: To display a variable called userName, use {{ common }}.
    • Arithmetic: You can perform simple math like {{ curly }}.
    • Methods: You can call component methods, such as {{ methods }}.
    • String Methods: Use JavaScript string functions like {{ curly }}.
  2. Working with Objects

      3. You can access object properties directly within the curly braces.

    1. Dot Notation: Use {{ dotnot }} to display a property.
    2. Safe Navigation Operator (?.): Use this to guard against null or undefined values. For example, {{ dotnot2 }} will safely return an empty string if user or profile is missing, instead of throwing an error.
    3. JSON Pipe: To debug and see the entire object structure on the page, use the JSON Pipe: {{ jpipe }}.

Rules and Constraints

  • The expression must return a value (string, number, boolean, object.property)
  • You CAN call pure methods and use pipes
  • You CANNOT use assignments (=), increment (++), or new keyword
  • You CANNOT access global objects like window or document directly
  • Expressions are re-evaluated every change detection cycle

Examples

Here is an example showing how to access properties from objects and elements from arrays using string interpolation in an Angular Component.

A — String Interpolation {{ curly }}

Title: {{ title }}

Student: {{ studentName }} | Course: {{ courseCode }}

Next year: {{ currentYear + 1 }}

Status: {{ getStatus() }}

Active: {{ isActive ? 'ACTIVE' : 'INACTIVE' }}

Name: {{ name }}

Course: {{ course }}

Year Level: {{ year }}

Next Year Level: {{ year + 1 }}

Student Info: {{ name + ' | ' + course + ' - Year ' + year }}

Year Status: {{ year >= 3 ? 'Upperclassman' : 'Underclassman' }}

Name (Uppercase): {{ name.toUpperCase() }}

B — Interpolating Objects

User Name: {{ user.name }}

User Position: {{ user.position }}

User Department: {{ user.department }}

User Role: {{ user.details.role }}
